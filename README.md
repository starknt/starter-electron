## starter-electron

基于 `Vite` 和 `Esbuild` 用于快速构建 Electron 应用的模板。

## 特性

- 🚀 极速启动
- ⚡ 主进程快速重载
- ✈ 多平台构建
- 🎉 减少构建产物体积(PS: 自动清理 `Native Module` 的无效文件)
- 🍛 双 `package.json` 结构, 分离原生模块和非原生模块
